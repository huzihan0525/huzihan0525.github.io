What is Machine Learning

Machine Learning: A computer program is said to learn from experience E with respect to some class of tasks T and performance measure P, if its performance at tasks in T, as measured by P, improves with experience E – Mitchell

由此可见，机器学习便是通过经验来完善系统自身的性能。根据不同的任务T，不同的性能评估方法P，不同的训练数据(经验)E，可将机器学习分为很多类。而采用概率的观点的优势则是可以在存在不确定性的情况下完成最优决策。
机器学习大致可分为 监督学习，非监督学习，强化学习。

Supervised Learning 监督学习

  • Task T: 学习一个映射(mapping) ，其中输入称作features, covariates, predictors，exogenous variable. 输出称作label, target, response, endogenous variable
  • Experience E: 输入-输出对 , 称作训练集(training set)，N为样本量
  • Performance Measure P: 取决于输出的种类
    根据不同的输出结果，我们可以把监督学习分为 Classification和 Regression.

Design Matrix

对于较小的 fixed-size feature vectors的数据集，我们常常使用的 design matrix来存储数据特征，每一行代表一个实例，每一列代表一个特征，当时，数据集成为 big data，当时，数据集成为 wide data.
在python中，可以使用pandas库中的DataFrame来构造 design matrix。

对于variable-size feature vectors，我们常常使用 featurization 将其转化为等长特征表示。

使用探索性数据分析EDA来分析数据，可以发现数据有无明显的模式或是否存在问题时。

Overfitting and Generalization 过拟合和泛化

过拟合指模型过于复杂，假设过于严格，对训练数据具有较为完美的拟合，为验证是否发生过拟合。假设已知分布，计算在真实分布的理论期望误差(theoretically expected loss/ population risk).

称为泛化误差(generalization gap)，实际情况真实分布未知，可以运用test set 计算出的 test risk代替 population risk，我们要求的模型是在测试集上损失最小的。如图，我们会选择D=2的模型。


除此之外，还需要验证集(validation set)来进行模型选择(model selection)

没有免费午餐定理: 没有最好的模型，所有的模型都是有效的。任何模型对所有真实问题（假设真是问题f均匀）的误差期望相同。因此需要根据具体的问题来选择算法\模型。
