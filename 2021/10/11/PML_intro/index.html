之前断断续续学过一些比较基础的机器学习的知识，也零零散散听过一些网课和著名的国内教材西瓜书以及李航教授的统计学习方法，但大都没有看完，或者未能应用实践。希望这次可以在理论与应用上对这部分知识进行系统化的学习，并通过此博客来记录一些学习笔记。但本人水平有限，若存在错误，恳请大家谅解与指明，不胜感激，期待与大家的交流。

这次学习参考书是Google研究员Kenvin P. Murphy《Probabilistic Machine Learning》，这本书是大名鼎鼎的MALPP《Machine Learning: A Probabilistic Perspective》的第二版，希望自己可以坚持读下去。本教材的特点是运用概率模型和贝叶斯决策理论贯穿全书。

What is Machine Learning

Machine Learning: A computer program is said to learn from experience E with respect to some class of tasks T and performance measure P, if its performance at tasks in T, as measured by P, improves with experience E – Mitchell

由此可见，机器学习便是通过经验来完善系统自身的性能。根据不同的任务T，不同的性能评估方法P和不同的训练数据(经验)E，可将机器学习分为很多类。而采用概率的观点的优势就是可以在存在不确定性的情况下完成最优决策。

Supervised Learning 监督学习

Task T: 学习一个映射(mapping) ，其中输入称作features, covariates, predictors，输出称作label, target, response.
Experience E: 输入-输出对 , 称作训练集(training set)，N为sample size
Performance Measure P: 取决于输出的种类
